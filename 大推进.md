==================================================
一、移动填报必须做成独立端，而不是主系统里的一个普通页面分区
==================================================

【目标】
面向一线数据输入角色的登录后工作界面，必须是一个独立端：
- 只服务主操、班组长、专项 owner 等数据输入角色
- 登录后只能看到填报相关页面
- 不可看到审阅、治理、成本、运维、主数据配置等后台能力
- 风格上与主系统统一，但信息架构、导航、壳层完全独立
- 可以单独部署到 `/entry` 或独立子域，但本轮优先在同仓库完成“单仓多壳”的独立端实现

【实现原则】
1. 不拆后端，不另起一套接口体系。
2. 基于现有 `/mobile/*` 能力升级实现。
3. 新建独立壳层，例如：
   - `frontend/src/layout/EntryShell.vue`
   - `frontend/src/views/entry/*`
4. 路由上形成独立端入口，例如：
   - `/entry`
   - `/entry/report/:businessDate/:shiftId`
   - `/entry/advanced/:businessDate/:shiftId`
   - `/entry/ocr/:businessDate/:shiftId`
   - `/entry/anomaly`
   - `/entry/history`
   - `/entry/drafts`
5. 旧 `/mobile/*` 路径继续兼容，并 301/302 或前端 redirect 到新独立端路径。
6. 对 fill-only 角色做强权限限制：
   - 登录后默认落地到独立填报端
   - 访问 review / desktop 时一律重定向回 `/entry`
   - 不显示任何审阅/管理/运维导航
7. 允许保留“历史记录 / 草稿箱 / 提交成功页 / 拍照识别 / 异常补录”这些输入辅助能力。

==================================================
二、成本核算必须做成“可配置策略引擎”，不能只做静态页面
==================================================

【总体核算原则】
成本核算统一遵循：
- 按班次归集
- 按实耗计价
- 按产量或通货量折算吨耗
- 按日汇总
- 按月滚动累计
- 分车间汇总、总表校差

统一公式抽象为：
- 直接成本 = Σ(数量 × 单价) + Σ(金额直录项目)
- 能源成本 = Σ(电/气/水/蒸汽/空压等消耗 × 单价)
- 人工成本 = 固定人工总额 或 岗位道次/分钟规则计算
- 设备保障成本 = 油品/辊系/热电偶/刀具/维护材料等合计
- 工序总成本 = 直接成本 + 能源成本 + 人工成本 + 设备保障成本 + 包装辅材 + 公辅分摊
- 产量吨耗 = 工序总成本 / 产量
- 通货吨耗 = 工序总成本 / 通货量
- 月累计吨耗 = 月累计总成本 / 月累计产量（或通货量）

【你必须实现的成本策略类型】
1. `CASTING_MACHINE_LABOR_SPLIT`
   - 适用：铸二、铸三
   - 逻辑：机列直接成本 + 人工分摊
   - 电费 0.8 元/度
   - 天然气 3.6 元/立方
   - 合金与辅材按价格主数据计入
   - 综合吨耗 = (直接成本 + 人工总额) / 当日产量

2. `FINISHING_PARALLEL_PROCESS`
   - 适用：精整
   - 工序：新19辊、纵剪、包装
   - 各工序分别归集人工、辅料、电费、包装费
   - 工序吨耗 = 工序总成本 / 工序产量
   - 综合吨耗 = 各工序汇总
   - D40 单价可先固化为 9.5 元/公斤
   - 其他包装材支持“金额直接入账”与“价格主数据”双模式

3. `HOT_ROLLING_SHIFT_DAILY`
   - 适用：热轧
   - 按白班/小夜/大夜录入
   - 人工按 178 元/人计，或兼容直接金额
   - 水 1.1 元/吨
   - 电 0.8 元/度
   - 天然气 3.6 元/立方
   - 支持油品、刀具、包装辅材、设备保障费用
   - 日总吨耗 = 日总成本 / 日产量
   - 前端展示中需支持示例解释：例如 4 月 15 日产量 410.93 吨、总成本 136085.37 元、日总吨耗 331.16 元/吨

4. `TENSION_LEVELING_MAIN_PLUS_AUX`
   - 适用：拉矫
   - 工序：退火炉、拉矫、大分切、包装
   - 退火炉：电 0.8 元/度、天然气 3.6 元/立方、热电偶 17 元/米
   - 拉矫主线：水 4 元/吨、铝套筒 5 元/公斤、D40 9.5 元/公斤、钢带 4.1 元/公斤、钢带扣 4.37 元/公斤
   - 空压机用电、锅炉房天然气单列，并支持分摊规则
   - 输出必须同时显示工序吨耗、公辅吨耗、综合吨耗

5. `LOSS_DUAL_CALIBER`
   - 适用：2050、1650、1850、花纹板
   - 逻辑：先汇总损耗金额，再同时除以通货量与产量形成双口径单耗
   - 电费 0.8 元/度
   - 轧制油 8.2 元/公斤
   - 白土 39 元/袋
   - 硅藻土 54 元/袋
   - 辊系保障费用 2000 元/次
   - 花纹板额外支持：辊系保障费用 6000 元/次、配辊 1000 元/次、飞滤素 9 元/公斤
   - 人工不是按人数平均，而是按岗位道次 + 分钟计费
   - 2050 规则：
     - 主操 11.2 元/道、0.42 元/分钟
     - 副操 6.7 元/道、0.31 元/分钟
     - 普工 5.6 元/道、0.26 元/分钟
     - 行车工 4.5 元/道、0.23 元/分钟
   - 1650 规则：
     - 主操 12.9 元/道、0.42 元/分钟
     - 副操 7.7 元/道、0.31 元/分钟
     - 普工 6.5 元/道、0.26 元/分钟
     - 行车工 5.2 元/道、0.23 元/分钟
   - 1850 若当前规则不完整，先支持人工金额直录 + 后续规则扩展
   - 输出必须支持“按通货量 / 按产量”口径切换
