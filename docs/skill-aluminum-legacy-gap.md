# Skill: aluminum-legacy-gap

## Purpose

Review legacy manual statistics assets and map them to current `aluminum-bypass` system surfaces.

## Workflow

1. Classify each legacy file by business kind.
2. Extract sheet names, header patterns, and sample rows.
3. Separate main workflow facts from observer-side sidecar facts.
4. Produce:
- absorbed by system
- partially absorbed
- not yet modeled
5. Recommend the smallest structural changes first.

## Output

- Findings
- Risks
- Minimum implementation plan
