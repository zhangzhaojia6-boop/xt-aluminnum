# 排班导入样例说明

样例文件: `docs/import-templates/schedule_sample.csv`

字段说明:
- `employee_no`: 员工工号，优先用于匹配员工
- `schedule_date`: 排班业务日期，格式 `YYYY-MM-DD`
- `shift_code`: 班次编码，需与 `shift_configs.code` 一致（如 `A/B/C`）
- `team_code`: 班组编码，用于覆盖员工默认班组（可选但建议填写）
- `workshop_code`: 车间编码，用于覆盖员工默认车间（可选但建议填写）

最小成功条件:
- 文件格式为 `csv` 或 `xlsx`
- 至少包含上述字段
- `employee_no` 在员工主数据中存在
- `shift_code` 在班次配置中存在
