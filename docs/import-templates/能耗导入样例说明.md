# 能耗导入样例说明

## 模板用途
用于导入每日能耗记录，系统会按业务日期、车间、班次汇总电/气/水等能耗，并计算单吨能耗。

## 文件格式
- 支持 `csv` / `xlsx`
- 编码建议 `UTF-8`

## 字段说明
| 字段 | 必填 | 说明 |
| --- | --- | --- |
| business_date | 是 | 业务日期，格式 `YYYY-MM-DD` |
| workshop_code | 是 | 车间编码 |
| shift_code | 是 | 班次编码 |
| energy_type | 是 | 能耗类型：`electricity`/`gas`/`water`/`other` |
| energy_value | 是 | 能耗数值 |
| unit | 否 | 单位，例如 `kwh`、`m3` |
| source_row_no | 否 | 原始行号 |
