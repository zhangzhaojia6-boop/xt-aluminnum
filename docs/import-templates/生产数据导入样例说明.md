# 生产数据导入样例说明

样例文件：`docs/import-templates/production_sample.csv`

## 字段定义（最小）
- `business_date`：业务日期，格式 `YYYY-MM-DD`
- `shift_code`：班次编码，需在 `shift_configs` 中存在
- `workshop_code`：车间编码，需在 `workshops` 中存在
- `team_code`：班组编码（可空）
- `equipment_code`：设备编码（可空）
- `input_weight`：投料重量
- `output_weight`：产出重量
- `qualified_weight`：合格重量
- `scrap_weight`：废品重量
- `downtime_minutes`：停机分钟数
- `issue_count`：问题数
- `electricity_kwh`：电耗（kWh）
- `notes`：备注

## 导入规则（第三轮基础版）
1. 支持 `csv` / `xlsx`
2. 同一 `business_date + shift_code + workshop_code + equipment_code` 不允许重复有效记录
3. 导入时自动计算：
   - `actual_headcount`：按考勤结果聚合
   - `planned_headcount`：优先用导入值，否则用排班人数
4. 导入后自动识别生产异常并写入 `production_exceptions`
5. 导入结果写入 `import_batches` 和 `import_rows`
