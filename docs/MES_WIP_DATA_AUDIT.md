## 待 MES 方确认事项

1. **车间编码** — 当前使用截图中的中文名称，需确认 MES 系统内部车间编码
2. **工序编码** — 当前使用截图中的中文名称，需确认 MES 系统内部工序编码
3. **日期口径** — 当日投料量的统计起止时间（自然日 vs 班次日）
4. **在制料计算口径** — 在制料总计的计算方式（投入未产出 vs 库存快照）
5. **"-" 语义** — 截图中 "-" 表示 0 还是无数据（当前按 null 处理）
6. **API 字段名** — MES 方提供的 API 字段命名规范
7. **刷新频率** — 数据更新周期（实时 / 小时 / 班次 / 日）
8. **权限范围** — 哪些角色可查看在制料数据
9. **数据格式** — 重量单位（kg vs t）、精度要求
